# NovarelStudio.com Design Guidelines

## Design Approach: Reference-Based Hybrid

Drawing inspiration from OnlyFans (aesthetic), Twitch/Kick (gamification), and Opus.pro/Vizard.ai (dashboard functionality) to create a unique AI content platform identity.

## Core Design Elements

### Typography
**Primary Font**: Inter Extra Bold (blocky, bold, Twitch/Kick style)
- Hero Headlines: 48px-64px, font-weight: 800, letter-spacing: -0.02em
- Section Headers: 32px-40px, font-weight: 800, letter-spacing: -0.02em
- Subheadings: 24px-28px, font-weight: 700

**Secondary Font**: Inter (for dashboard, UI, body text)
- Dashboard Headers: 24px-28px, font-weight: 600-700
- Tab Labels: 14px-16px, font-weight: 600
- Body Copy: 14px-16px, font-weight: 400-500

### Layout System
**Spacing Units**: Tailwind spacing - 4, 6, 8, 12, 16, 24, 32
- Container: max-w-7xl with px-6 padding
- Section Padding: py-20 (desktop), py-12 (mobile)
- Card Spacing: gap-6 to gap-8
- Grid Layouts: 3 columns (desktop), 2 columns (tablet), 1 column (mobile)

### Component Library

**Navigation**
- Sticky header with blur backdrop (similar to OnlyFans clean header)
- Logo (cursive font) + primary nav items + CTA button
- Dashboard tabbed navigation (Opus.pro style): My Projects | Analytics | Settings | AI Clipping

**Hero Section**
- Full viewport height (min-h-screen) with blue gradient background
- Large cursive headline with light font weight
- Animated live stats counter showing real-time metrics (Twitch-style)
- Primary CTA button with subtle glow effect
- Hero image: Abstract AI/tech visualization or streamers in action

**Gamification Elements** (Twitch/Kick inspired)
- Live viewer counters with pulsing animations
- Achievement badges (circular with neon borders)
- Progress bars with gradient fills
- XP-style level indicators
- Neon accent highlights (purple/cyan accents on blue base)
- Streaming platform icons with glow effects

**Dashboard Interface** (Opus.pro/Vizard.ai style)
- Left sidebar navigation with icon + label
- Main content area with tabbed sections
- Video thumbnail grid (3-4 columns)
- Filter/sort controls at top
- Action buttons (Upload, Generate Clips, Publish)

**Feature Cards**
- Grid layout (3 columns desktop)
- Gaming-style icons (circular with gradient backgrounds)
- Title + description + "Level up" style progression indicators
- Hover effect: lift + glow shadow

**Video Showcase Section**
- Before/After comparison slider
- Video thumbnails with play button overlays
- Platform badges (TikTok, YouTube, Instagram)
- Engagement metrics display (views, likes simulated)

**Settings Panel**
- Organized tab structure: Account | Streaming Platforms | AI Preferences | Billing
- Form inputs with rounded corners (border-radius: 12px)
- Toggle switches with blue accent
- Save/Cancel buttons with hover states

**Platform Integration Display**
- Connected platforms grid
- Status indicators (online/offline with pulsing dots)
- Animated connection lines between elements
- Real-time metrics counters

**Forms & Inputs**
- Rounded input fields (rounded-xl)
- Blue focus rings
- Floating labels
- Primary button: blue background, white text, rounded-full
- Button hover: slight scale + brightness increase

**Stats & Analytics**
- Card-based metrics layout
- Circular progress indicators
- Line graphs with gradient fills
- Achievement-style milestone markers

**Call-to-Action Sections**
- Large, centered CTA with cursive headline
- Waitlist signup form with elegant styling
- Social proof elements (user count, testimonials)

### Images
**Hero Image**: Large, full-width abstract AI technology visualization or streamer action shots with blue overlay gradient

**Feature Section Images**: Screenshots of AI-generated clips, dashboard interface, platform integrations

**Video Thumbnails**: Placeholder gaming/streaming content with play overlays

**Background Elements**: Subtle geometric patterns, gradient meshes, particle effects (minimal, not distracting)

## Page Structure

**Landing Page Sections**:
1. Hero (full viewport with stats animation)
2. Trusted By / Social Proof
3. How It Works (5-step pipeline with level progression)
4. Feature Showcase (video examples grid)
5. Platform Integrations
6. Analytics Preview
7. Pricing/Waitlist
8. FAQ
9. Footer

**Dashboard Sections**:
1. Header with user profile
2. Sidebar navigation
3. Main content area with tabs
4. Project grid
5. Quick actions panel

## Key Design Principles
- **Blue & White Dominance**: Clean, professional aesthetic from OnlyFans
- **Cursive Elegance**: Lighter, more flowing than OnlyFans typography
- **Gamified Engagement**: Twitch/Kick-style live stats, badges, progress indicators
- **Professional Dashboard**: Opus.pro/Vizard.ai-inspired organization and functionality
- **Smooth Animations**: Subtle transitions, hover effects, progress animations
- **Responsive Excellence**: Mobile-first with breakpoint optimizations
